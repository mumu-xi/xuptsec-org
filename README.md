# xuptsec-org
 ## start
 ```
   git clone git@github.com:mumu-xi/xuptsec-org.git
 ```
## install
```
   cd xuptsec-org/client // 前端目录
   npm install
   npm start
   http://localhost:8080

   cd ../server // 后端目录
   npm install
   npm start // 开启服务器
   http://localhost:8181

```
## 部署与打包
```
   npm run build
```
