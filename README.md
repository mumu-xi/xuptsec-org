# xuptsec-org
 ## start
 ```
   git clone git@github.com:mumu-xi/xuptsec-org.git
 ```
##install
   cd xuptsec-org/client //前端目录
   npm install
   npm start

   cd ../server // 后端目录
   npm install
   npm run server // 开启服务器
